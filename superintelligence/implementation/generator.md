basically generates different kinds of the same text 

so for example:
_Alex_ went to the _bank_ and _withdrew_ some _money_

each of the words in "_italics_" are variables that can be alterred with other names/lists etc. 

there also exists an ambiguity knob, which helps decide whether the ambiguity in that sentence should be at 1 (near 0 ambiguity) or 0 (max ambiguity) so we can test training on ambiguous sentences
